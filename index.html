Informacion sobre IKlan

IKlan fue creado por curiosidad a NodeJS, tambien se utiliza como metodo de aprendizaje, tomo el metodo de Iklan para crear proyectos aprendiendo HTML, CSS y JS .

La unica funcion de IKlan es dar Anuncios en tu servidor de Discord mediante el bot, tambien cuenta con unos pocos comando sencillos.

El pricipal es k!anuncio pero hay otros mas como:
k!user
k!say
k!poll
